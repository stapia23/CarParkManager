# CarParkManager

A mobile web app to help car parking lots manage inventory based on vehicle size, arrival date, and pickup schedule. Built using React Native and Firebase.

## Features
- Real-time vehicle tracking
- Optimized vehicle placement
- Inventory visualization and reports

## Tech Stack
- **Frontend:** React Native
- **Backend:** Firebase (Authentication + Firestore DB)

## Start the app

   ```
    npx expo start
   ```
